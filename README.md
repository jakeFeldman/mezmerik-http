# @mezmerik/http

A JavaScript http request utility

## Getting Started

Install @mezmerik/http

### Installing

Clone the repository

with npm

```bash
npm install @mezmerik/http
```

with yarn

```bash
yarn add @mezmerik/http
```

Start the dev server

```bash
yarn start
```
